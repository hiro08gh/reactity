# reactity

> reactity is modern React template.

## Features

The following features are currently included:

- [x] React + TypeScript
- [x] Unit test using jest and react-testing-library
- [x] E2E test using Cypress

## Getting started

The primary you will need to install package:

```
$ yarn install
```

Start server

```
$ yarn start
```

build

```
$ yarn build
```

## E2E test

```
$ yarn run e2e
```
